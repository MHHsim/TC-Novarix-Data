# Proyecto E-Commerce: Novarix Data
Proyecto Team Challenge - M004
## Descripción
Este proyecto consiste en el diseño y despliegue de una infraestructura de datos completa para un E-Commerce, desde el modelado relacional (3NF) hasta la implementación en Google BigQuery.

## Equipo
* **Scrum Master:** Marta
* **Data Architect:** Claudia
* **Data Engineer:** Isaac
* **QA & Documentation:** María

## Stack Tecnológico
* **Base de Datos:** Google BigQuery
* **Lenguaje:** Python 3.x (entorno virtual `venv`)
* **Librerías:** Pandas, Faker, Google-cloud-bigquery

## Estructura del Repositorio
* `notebooks/01_setup_bigquery.ipynb`: Creación de tablas.
* `notebooks/02_generate_data.ipynb`: Poblado de datos con Faker.
* `notebooks/03_queries_verification.ipynb`: Validaciones de QA.
* `docs/`: Diagrama Entidad-Relación.
