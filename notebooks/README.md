# Guía de Notebooks

En esta carpeta se encuentran los procesos de datos del proyecto Novarix.

### Orden de ejecución:
1. **01_setup_bigquery.ipynb**: Configuración de tablas y esquemas en BigQuery. ESTE NO ESTÁ
2. **02_data_generation.ipynb**: Generación de datos sintéticos usando Faker. ESTE TAMPOCO
3. **03_queries_verification.ipynb**: Notebook de QA para verificar la integridad y calidad de los datos generados. ESTE ESTÁ PERO VACÍO
